# Метео-планировщик асфальтоукладки

> Хакатон **«Цифровая Эра Транспорта 2026»** — кейс Ассоциации ЦЭТ.
> Приложение планирует работы по укладке асфальтобетонного покрытия в зависимости
> от прогноза погоды: рассчитывает «зелёные окна», управляет логистикой смеси,
> перенаправляет самосвалы и автоматически выдаёт наряды на ТО при непогоде.

## Что делает приложение

1. **Агрегирует** прогноз погоды из 3+ независимых источников и снимает противоречия.
2. **Считает «зелёное окно»** — интервал времени без осадков с учётом доставки смеси
   от АБЗ до участка и температурных ограничений (T возд ≥ +5 °C, T смеси ≥ 140 °C
   при старте уплотнения, ≥ 80 °C к концу).
3. **Калькулятор «Успеть до дождя»** — максимальный тоннаж, который реально уложить
   до наступления осадков.
4. **Динамически перенаправляет самосвалы**: если на участке А дождь, ищет сухой
   участок Б в радиусе досягаемости и перенаправляет загруженную смесь туда.
5. **Планировщик ТО**: при подтверждении осадков формирует наряды на осмотр
   простаивающих укладчиков и катков.
6. **Подвоз тех-смесей**: при прогнозе заморозков — автозаказ антифриза, керосина
   (антиадгезионка), промывочных жидкостей.
7. **Карта мастера участка** (mobile-first): АБЗ, участки, техника, радар осадков,
   пуш-уведомления о закрытии «зелёного окна».

Полная постановка — в [`docs/source/case_brief.pdf`](./docs/source/case_brief.pdf)
и [`docs/source/implementation_plan.pdf`](./docs/source/implementation_plan.pdf).

## Быстрый старт для разработчика

```bash
# 1. Клонируем
git clone <repo-url> && cd Moskva_Case-in

# 2. Поднимаем БД и сервисы
cp .env.example .env
docker compose -f infra/docker-compose.yml up -d db

# 3. Backend
cd backend
python -m venv .venv && source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000

# 4. Frontend (в другом терминале)
cd frontend
npm install
npm run dev   # http://localhost:5173
```

API-документация Swagger: <http://localhost:8000/docs>.

## Ключевые формулы (из ГОСТ Р 58406.2-2020 и тех. карты)

**Производительность укладки**, м²/ч (формула 1 техкарты):

```
ПУ = ТУ · КВ · a · V / 8
```

**Скорость укладчика** из условия снабжения смесью (формула 2):

```
V = П_абз / (a · h · ρ · 60)
```

где `П_абз` — производительность АБЗ, т/ч; `a` — ширина полосы, м;
`h` — толщина слоя, м; `ρ` — плотность смеси, т/м³.

**Температурные пороги уплотнения**:

| Этап | Каток | T поверхности, °C |
|---|---|---|
| Предварительное | 9 т | 145 → 120 |
| Основное | 12 т | 120 → 100 |
| Заключительное | 12 т | 100 → 80 |
