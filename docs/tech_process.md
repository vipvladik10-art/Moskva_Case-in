## 5. Формулы

### 5.1 Производительность укладки (формула 1)

```
ПУ = ТУ · КВ · a · V / 8         [м²/ч]
```

- `ТУ` — продолжительность рабочей смены, мин;
- `КВ` — коэффициент использования = **0,8** (безразмерный);
- `a` — ширина полосы, м;
- `V` — скорость движения укладчика, м/мин.

### 5.2 Скорость укладчика из условия снабжения (формула 2)

```
V = П_абз / (a · h · ρ · 60)     [м/мин]
```

- `П_абз` — производительность АБЗ, **т/ч**;
- `a` — ширина полосы, м;
- `h` — проектная толщина слоя, м;
- `ρ` — плотность смеси, **т/м³**.

### 5.3 Максимальный тоннаж «Успеть до дождя» (наша формула)

Если `t_окно` — длительность зелёного окна (часов) и есть запас на доставку
`t_дост`, то максимальный безопасный заказ:

```
T_max = min(
    П_абз · (t_окно - t_дост),                          # ограничение АБЗ
    a · h · ρ · V · 60 · (t_окно - t_дост)              # ограничение укладчика
)                                                        [т]
```

Дополнительно вычитаем смесь, которая «не успеет уплотниться до 80 °C»
по таблице 1 (см. ниже).

## 8. Прочие критичные параметры

- **Заказ смеси на АБЗ**: минимум за **4 часа** до укладки.
  Это «магическое окно прогноза» — именно здесь приложение даёт максимум пользы:
  оно должно предсказать осадки на горизонте 4–6 ч с точностью лучше
  человека-мастера, смотрящего Яндекс.Погоду утром.
- **Прогрев плиты укладчика**: 10–30 мин (учитывать в `t_дост`).
- **Стояние перед стартом**: 3–6 мин с загруженной смесью.
